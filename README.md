## 配置需求
最低需求:1v核心2GB内存 40Gb可用空间 10Mbps网络带宽 Ipv4公网ip一个 UDP端口正常开放
推荐需求:4v核心8GB内存 60Gb可用空间 50Mbps网络带宽 Ipv4公网ip一个 UDP端口正常开放
